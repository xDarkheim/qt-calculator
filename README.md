# Qt Calculator

A simple calculator application built with Qt5 and C++20.

## Features
- Basic arithmetic operations (+, -, *, /)
- Clear functionality
- Error handling for division by zero
- Modern C++ practices (Rule of Five)

## Build Requirements
- Qt5 (Core, Widgets)
- CMake 3.16+
- C++20 compatible compiler

## Building
```bash
mkdir build && cd build
cmake ..
make
